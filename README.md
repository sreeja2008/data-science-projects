# Python Projects

This repository contains beginner Python projects developed for learning and practice.

## Projects

### Student Grade System

* Calculates student grades based on marks.
* Generates grade reports and performance analysis.

### Password Validator

* Checks password strength and validation rules.
* Demonstrates conditional statements and string handling.

### Login System

* Authenticates users using username and password.
* Reads credentials from a file.
* Logs login attempts with timestamps.
* Uses exception handling for error management.

### Voting System

* Collects votes for multiple candidates.
* Stores voting data using dictionaries.
* Determines the winner based on vote count.
* Handles tie situations between candidates.

## Technologies Used

* Python
* Dictionaries
* File Handling
* Exception Handling
* Loops
* Conditional Statements

## Learning Outcomes

* Problem Solving
* Data Validation
* User Authentication
* File Operations
* Dictionary Usage
